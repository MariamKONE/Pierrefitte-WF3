SELECTIONNER DES ELEMENTS DE LA PAGE HTML AVEC JS

le DOM (Document Object Model)-

1/ document.getElementById()

Recherche dans l'arbre DOM d'une balise ayant l'id spécifié.

document.getElementById() renvoie un objet DOM manipulable en JavaScript correspondant à la balise trouvée.

Sinon la méthode renvoie la valeur spéciale null.

2/ document.getElementsByClassName()

Recherche dans l'arbre DOM de toutes les balises ayant la class spécifiées.

document.getElementById() renvoie un objet DOM manipulable en JavaScript correspondant à la balise trouvée.

Sinon la méthode renvoie la valeur spéciale null.

3/ document.getElementsByTagName()

Recherche dans l'arbre DOM de toutes les balises ayant la balise spécifiée.

document.getElementById() renvoie un objet DOM manipulable en JavaScript correspondant à la balise trouvée.

Sinon la méthode renvoie la valeur spéciale null.

4/ document.querySelector()

Recherche dans l'arbre DOM de la 1ere balise rencontrée qui a l'id, la class ou le tag spécifié.

document.getElementById() renvoie un objet DOM manipulable en JavaScript correspondant à la balise trouvée.

Sinon la méthode renvoie la valeur spéciale null.

Modifier le contenu d'une balise

  • textContent = ''; => remplace le contenu de la balise par du texte
  • innerHTML = ''; => remplace le contenu de la balise par de l'HTML

AJOUTER OU SUPPRIMER DES CLASS D'UN ELEMENT

  • elt.classList.add('uneclass');

    Ajoute une class à l'élément sélectionné

  • elt.classList.remove();

    Remove une class à l'élément sélectionné

  • elt.classList.toggle();

    Ajoute une classe si elle n'existe pas et la retire si elle existe à l'élément
