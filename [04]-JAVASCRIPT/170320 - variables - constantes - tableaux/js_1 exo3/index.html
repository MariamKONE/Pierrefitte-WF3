Exo 3 : Calculatrice HT vers TTC

L'utilisateur saisit un montant HT, le résultat TTC s'affiche en HTML.

1- le taux de TVA utilisé est le taux normal en vigeur en France.
2- le taux de TVA n'est pas variable, il faut donc utiliser une constante.
3- afficher toutes les informations : montant HT, montant de TVA et résultat TTC.
4- attention au type de donnée renvoyé par window.prompt()
5- le montant HT peut être à virgule.
